# 🐢 Sea Turtle ESA Analysis

## Overview
This project analyzes how the Endangered Species Act impacts sea turtle populations.

## Species Included
- Green Sea Turtle
- Loggerhead Sea Turtle
- Leatherback Sea Turtle

## Key Insight
Different species respond differently to protection. Some recover quickly, while others continue to decline.

## Tools Used
- Python
- Pandas
- Matplotlib
- Streamlit

## How to Run
streamlit run app.py
